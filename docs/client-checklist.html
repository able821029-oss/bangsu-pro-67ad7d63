📌 먼저 알아두실 것 — "SaaS 앱"이란?

매달 이용료를 받고 고객이 인터넷으로 접속해 사용하는 앱입니다. 넷플릭스나 쿠팡이츠처럼 "매달 자동결제"되는 구조입니다.

출시하려면 크게 4가지가 필요합니다:

  1. 열쇠(API 키) — 외부 서비스 접속용 비밀번호
  2. 돈 받는 통로(결제) — 카드사 연결
  3. 법적 서류 — 법으로 꼭 있어야 하는 문서
  4. 브랜드 자료 — 로고, 색깔, 소개글

🗝️ 1. 열쇠(API 키) — 외부 서비스 접속 비밀번호

앱이 혼자 작동하는 게 아니라 다른 회사 서비스를 빌려서 작동합니다. 각 회사의 "열쇠"가 필요합니다.

① AI 글쓰기 열쇠 🤖

  • 뭐예요? ChatGPT 만든 회사(OpenAI)의 열쇠
  • 왜 필요해요? 블로그 글 자동 생성 기능용
  • 받는 법:
    1. platform.openai.com 가입
    2. 신용카드 등록
    3. "API Keys" 메뉴에서 키 생성
    4. sk-...로 시작하는 문자열 전달
  • 비용: 사용한 만큼 과금 (월 5~30만원 예상)
⚠️ 월 한도 설정 필수 — 안 하면 사고 시 수백만원 과금 위험

② 영상 만드는 열쇠 🎬

  • 나레이션 목소리: ElevenLabs 또는 구글 TTS 계정
  • 배경음악(BGM): 상업용 사용 가능 음원인지 확인 필수!
    • 유튜브 무료 음악 → 절대 안 됨 (저작권 소송 위험)
    • 추천: Artlist, Epidemic Sound

③ 데이터베이스 열쇠 💾

  • 뭐예요? Supabase에 고객 정보/글/영상 보관 중
  • 필요한 것: 개발용 계정을 클라이언트 소유로 이전
  • 이메일 주시면 Owner 권한 넘겨드립니다
  • 비용: 처음 무료, 고객 많아지면 월 $25부터

④ 도메인(주소) 🌐

  • 뭐예요? sms.co.kr 같은 인터넷 주소
  • 어디서 사요? 가비아, 후이즈, 카페24 (1~2만원/년)
  • 결정 필요: 원하는 도메인 이름

💳 2. 결제 시스템 — 고객에게 돈 받는 통로

SaaS 앱은 매달 자동결제가 핵심입니다. 이게 없으면 출시 자체가 안 됩니다.

① 결제대행사(PG사) 선택 🏦

회사특징추천 대상
토스페이먼츠 ⭐가장 쉬움, 수수료 낮음국내 고객 위주라면
아임포트(포트원)여러 카드사 한번에카카오페이·네이버페이 중요시
Stripe해외 결제 강함외국 고객도 받을 경우

👉 추천: 토스페이먼츠

② 결제사 가입 서류 📄

⏰ 심사에 2~5영업일 걸리니 제일 먼저 준비하세요!
  • 사업자등록증 (홈택스 무료 신청, 1일)
  • 통신판매업 신고증 (구청 방문, 4만원, 1주)
  • 정산받을 통장 사본
  • 대표자 신분증 사본
💡 개인사업자도 가능합니다. 법인 아니어도 OK!

③ 요금제 정하기 💰 (가장 중요!)

이것만 정해주시면 바로 연결할 수 있습니다.

플랜월 요금제공 내용
브론즈? 원/월블로그 ?건, 영상 ?개
실버? 원/월블로그 ?건, 영상 ?개
골드? 원/월블로그 ?건, 영상 ?개

추가 결정 사항:

  • 무료 체험 줄까요? (예: 7일 무료)
  • 연간 결제 할인? (예: 1년 결제 시 2개월 무료)
  • 환불 규정? (예: 7일 이내 100% 환불)
  • 플랜 중간 변경 시 처리 방식 (일할 계산 / 다음달부터)

📜 3. 법적 서류 — 법으로 꼭 필요한 것

⚠️ 이거 없으면 출시 못 합니다. 개인정보보호법 위반입니다.

① 꼭 있어야 하는 3가지 📋

  1. 이용약관
    • 고객과 회사 사이의 규칙
    • 변호사 검토 추천 (20~50만원) 또는 템플릿 활용
  2. 개인정보처리방침
    • 어떤 정보를, 얼마나, 왜 모으는지 명시
    • 법적 의무
    • 개인정보보호위원회(privacy.go.kr)에서 무료 템플릿 제공
  3. 환불 규정
    • 전자상거래법상 의무
    • 예: "가입 후 7일 이내 미사용 시 전액 환불"

② 사업자 정보 공개 🏢 (앱 하단 표시 필수)

  • 상호 (회사 이름)
  • 대표자 이름
  • 사업자등록번호
  • 사업장 주소
  • 통신판매업 신고번호
  • 고객센터 전화번호
  • 고객센터 이메일
  • 운영 시간 (예: 평일 10~18시)

🎨 4. 브랜드 자료 — 앱의 얼굴

① 이미지 자료 🖼️

  • 로고 원본 (일러스트레이터 또는 고화질 PNG, 다크/라이트 2종)
  • 앱 아이콘 (1024×1024 PNG, 스마트폰 홈화면 아이콘)
  • 대표 이미지 (카톡/SNS 공유 썸네일, 1200×630)
💡 없으시면 디자인 작업도 같이 진행 가능합니다 (별도 비용)

② 글 자료 ✍️

  • 앱 한 줄 소개 (예: "사장님을 위한 자동 마케팅 앱")
  • 상세 설명 (3~5줄)
  • 고객센터 전화번호 / 이메일 / 카톡 채널

📧 5. 이메일 발송 시스템

고객이 가입/결제할 때 자동으로 이메일이 나가야 합니다.

  • 발송용 이메일 주소 결정 (예: noreply@sms-app.com)
  • 고객센터 이메일 별도 (예: help@sms-app.com)
  • 도메인 구매 후 저희가 설정 작업 진행

✅ 전체 작업 체크리스트 (순서대로!)

③ 도메인 구매가비아 등10분
④ 요금제 3가지 결정—1일 고민

🟢 3주차 — 모두 받으면

저희가 앱에 연결하고 실제 결제 테스트 후 정식 출시합니다! 🚀

📞 가장 먼저 답해주실 3가지

  1. 💰 요금제 3개 가격과 제공량 (브론즈/실버/골드)
  2. 🌐 원하는 도메인 이름 (예: sms-app.com)
  3. 📋 사업자등록증 유무 (있으면 전달, 없으면 신청 도와드림)

이 3가지만 주시면 나머지는 순서대로 알려드리면서 진행합니다!
