
# 올인원 마케팅/일정 관리 앱 구현 계획

기존 기능(AI 블로그 생성, 숏폼 영상, SEO 분석 등)을 유지하면서 아래 기능을 추가합니다.

## 1단계: 데이터베이스 & 인증 기반 구축
- **profiles 테이블** 생성 (user_id, name, email, google_token, terms_agreed 등)
- **schedules 테이블** 생성 (user_id, title, date, time, memo, image_url, google_synced)
- **contents 테이블** 생성 (user_id, text, image_url, created_at)
- **인증 설정**: 이메일 가입 + 구글 로그인 활성화
- **RLS 정책**: 각 테이블에 사용자별 데이터 접근 제어

## 2단계: 인증 UI
- 로그인/회원가입 페이지
- 구글 로그인 버튼 (Lovable Cloud 관리형)
- 이용약관 동의 체크박스 + 약관 페이지
- 인증 상태 관리 (AuthProvider)

## 3단계: 네비게이션 재구성
- 하단 탭 4개로 변경: 홈 / 일정(캘린더) / 콘텐츠 생성 / 마이페이지
- 기존 기능(카메라, AI글쓰기, 업로드, SEO, 숏폼 등)은 콘텐츠 생성 탭 내부에 배치

## 4단계: 캘린더/일정 관리 기능
- 캘린더 UI (월/주/일 보기)
- 일정 CRUD (등록/조회/수정/삭제)
- 플로팅 버튼(FAB)으로 일정 등록
- 일정 상세보기/수정 모달
- 이미지 첨부 기능

## 5단계: 구글 캘린더 연동
- Edge Function: 구글 캘린더 API 호출 (일정 생성)
- 일정 등록 시 자동 구글 캘린더 동기화
- 연동 상태 표시 ("캘린더 연동 완료" / "실패")
- **참고**: 사용자가 Google Cloud Console에서 Calendar API를 활성화하고, OAuth 클라이언트 ID/Secret을 설정해야 합니다

## 6단계: 마이페이지
- 프로필 정보 표시/수정
- 구글 캘린더 연동 상태 관리
- 기존 설정 기능 통합 (업체 정보, 구독 정보 등)
- 이용약관 보기

각 단계별로 구현하며, 1~2단계를 먼저 진행합니다.
