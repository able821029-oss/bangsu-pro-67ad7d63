## 방수PRO v2 리뉴얼 계획

### Phase 1: 구조 변경
1. **하단 탭 4개로 변경**: 홈 / 촬영 / 게시(중앙 원형) / 설정
2. **BottomNav 리디자인**: 게시 탭 중앙 원형 강조 버튼
3. **Index.tsx 업데이트**: WriterTab 제거, UploadTab → PublishTab

### Phase 2: 각 탭 리뉴얼
4. **홈 탭**: 업체 로고+이름, 요금제 잔여 건수, 빠른 실행 버튼 변경
5. **촬영 탭**: 플랫폼 선택(네이버/인스타/틱톡), 페르소나 선택(장인형/친근형/전문기업형) 추가
6. **게시 탭**: AI 생성 진행 + 미리보기 + 플랫폼별 탭 + 반자동 업로드 통합
7. **설정 탭**: 메뉴 목록 방식으로 변경 (프로필/요금제/쿠폰/소개/FAQ/문의/공지)

### Phase 3: 설정 하위 페이지들
8. **프로필 설정**: 로고 업로드, 업체명, 전화번호, 시공지역, SNS 연동, 토글
9. **요금제 페이지**: 4종 플랜 카드 비교, 연간 결제 할인
10. **쿠폰·혜택**: 보유 쿠폰 목록, 코드 입력
11. **지인 소개(레퍼럴)**: 소개 코드, 공유 버튼, 현황, 보상 구조
12. **FAQ**: 아코디언, 카테고리별 분류
13. **문의하기**: 유형 선택, 제목/내용 입력, 스크린샷 첨부, 문의 내역
14. **이탈 방지 팝업**: 해지 시 할인 제안, 사유 선택

### Phase 4: 스토어 업데이트
15. **appStore 확장**: 플랫폼 선택, 페르소나, 요금제, 쿠폰 등 상태 추가

> 참고: Claude API 연동, Stripe 결제, DB 연동은 UI 완성 후 별도 진행
> 현재는 모든 데이터를 목업/로컬 상태로 처리
